# Homebrew Tap for Bough

Official Homebrew tap for [Bough](https://github.com/DGPisces/bough).

## Install

```sh
brew tap DGPisces/tap
brew install --cask bough
```

Equivalent one-line form:

```sh
brew install --cask DGPisces/tap/bough
```

## Update

Homebrew manages updates for this install:

```sh
brew update
brew upgrade --cask bough
```

Bough's in-app updater is for the GitHub Releases DMG install path.

## Uninstall

```sh
brew uninstall --cask bough
```

To remove Bough user data as well:

```sh
brew uninstall --cask --zap bough
```
